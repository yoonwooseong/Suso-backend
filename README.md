# 수료증을 소개하다 Backend


#### 기술 스택
 - Language : `Java`
 - Library & Framework : `Spring-boot` `Spring-security` `JPA(hibernate)` `Swagger`
 - Database : `MySQL` `H2`
 - Target : `Web Browser(chrome)`

#### 버전
- `Java` : 11
- `Spring-boot` : 2.7.9
- `Spring-security` : 5.7.7
- `Springfox Swagger` : 3.0.0
